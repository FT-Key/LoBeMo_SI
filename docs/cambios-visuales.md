## CSS Global Updates

### Nuevas Animaciones Agregadas:
```css
✨ @keyframes gradient        - Fondo que cambia gradualmente
🎪 @keyframes float           - Elementos que flotan
💫 @keyframes glow            - Brillo pulsante
✨ @keyframes shimmer         - Efecto de brillo que pasa
↗️ @keyframes slideInUp/Down/Left/Right - Animaciones de entrada
🔍 @keyframes fadeInScale     - Fade + Scale combinados
📊 @keyframes pulse-soft      - Pulse suave personalizado
🎚️ @keyframes float-delay     - Float con control de delay
```

### Nuevas Clases de Utilidad:
```css
.animate-gradient        - Aplica animación gradient
.animate-float           - Aplica animación float
.animate-glow            - Aplica animación glow
.animate-shimmer         - Aplica shimmer effect
.animate-slide-*         - Deslizar en direcciones
.animate-fade-scale      - Fade + scale
.animate-pulse-soft      - Pulse suave
.delay-*                 - Delays para animations
```
