## Decisiones técnicas clave
- Hero-bg usa CSS background-image (no Next.js Image) para evitar problemas de optimización
- Multiply text usa background-clip: text + w-max + pointer-events-none para evitar clipping y click blocking
- Stacking: hero-bg (z-auto) → overlays (z-1) → content (z-10) → navbar (z-50)
- Text-stroke: 1.5px rgba(0,212,255,0.35) en todo el h1 para separar del fondo
- Middleware excluye extensiones estáticas (png|jpg|webp|svg|ico|css|js|woff2?|json)
- Sin gradientes en headings; colores sólidos para mejor degradado
