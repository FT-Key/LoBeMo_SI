## Description
Reporte de hallazgos de pentesting — sistema para registrar y gestionar hallazgos de pruebas de penetración.

## Acceptance Criteria
- AC-01: Registrar hallazgos de pentesting con título, descripción, severidad, estado y proyecto asociado.
- AC-02: Listar y filtrar hallazgos por severidad, estado o proyecto.
- AC-03: Ver detalle de cada hallazgo con toda su información.
