# US-017: Métricas y gráficos por proyecto

## Card Info
- **Trello Card**: https://trello.com/c/5IUiy7g1

- **Tamaño**: M (3-4 días)

## Descripción
**Como** CISO, **quiero** ver gráficos de avance, horas invertidas y tareas completadas por proyecto **para** evaluar el desempeño del equipo.

## Plan
1. API route: `/api/metricas/proyecto/[id]` — datos de tareas, estados y avance
2. Page: `/proyectos/[id]/metricas` — página de métricas con gráficos CSS
3. Components: barras de progreso y gráficos de barras con Tailwind
4. Nav link en detalle de proyecto

## Implementación
- API retorna: tareas por estado, tareas por prioridad, % completado, hitos
- Gráficos CSS sin librerías externas
- Acceso: CISO y GERENTE_GENERAL

## Quality Gates
- ✅ Build exitoso (TypeScript, compilación)
- ✅ Sin dependencias externas de gráficos
