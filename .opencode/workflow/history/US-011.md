### API
- **GET /api/notificaciones**: Returns `{ data: Notificacion[], unreadCount: number, pagination }`
- **POST /api/notificaciones**: Body `{ ids: string[] }` — marks all as read
- **PATCH /api/notificaciones/[id]**: Marks single as read
