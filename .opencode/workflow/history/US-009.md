### API (GET /api/dashboard)
- Query params: `desde`, `hasta` (ISO date strings)
- Response: `{ data: { proyectosPorEstado, statsEmpleados, ingresosDelMes, clientesNuevos } }`
- Roles permitidos: GERENTE_GENERAL, CISO, ADMINISTRACION

### Componentes
- **DashboardPage** (server): verifica rol, fetch inicial, renderiza DashboardContent
- **DashboardContent** (client): TanStack Query, filtro fechas, 4 widgets

### Widgets

2. Empleados ocupados vs disponibles — stats card con porcentaje
3. Ingresos del mes — monto total formateado
4. Clientes nuevos — contador con indicador de tendencia
