## Description
Como Gerente General, consultar el historial completo de cambios en el sistema para auditar quién hizo qué y cuándo.

## Acceptance Criteria
- AC-01: El AuditLog registra toda operación CREATE, UPDATE, DELETE (RN-09).
- AC-02: Se puede filtrar por entidad, empleado, fecha y acción.
- AC-03: Solo el Gerente General puede acceder al log completo.

Archivos a crear:

2. `src/app/auditoria/page.tsx` — Server component page con verificación de rol Gerente General
3. `src/components/audit-log/audit-log-list.tsx` — Client component con tabla + filtros + paginación

No se modifica schema ni otras rutas — el modelo AuditLog ya existe y todas las rutas ya registran logs.
