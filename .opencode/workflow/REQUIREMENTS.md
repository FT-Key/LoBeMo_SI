# Requirements: LoBeMo Seguridad Informática — Sistema de Gestión de Proyectos de Ciberseguridad

## Resumen Ejecutivo

LoBeMo Seguridad Informática es una empresa tucumana especializada en ciberseguridad, con 11 colaboradores organizados en 4 áreas (Gerencia, Administración, Comercial, Sistemas) más dos roles que reportan directamente a Gerencia (Auditoría y Capacitación).

**Stack:** Next.js 15 + Prisma + PostgreSQL (Neon) + Auth.js v5 + TanStack Query + Tailwind/shadcn/ui
**Arquitectura:** Hexagonal + DDD
**Deploy:** Vercel + GitHub

Ver archivo completo en la rama main para el detalle completo de 399 líneas con RF, RNF, RN y US.
